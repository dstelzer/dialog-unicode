# Dialog-Unicode

A demonstration of how you can customize [Dialog](https://github.com/Dialog-IF/dialog/)’s standard library. This one uses Unicode quotes, apostrophes, and dashes in place of the usual ASCII ones.

If you want to use this library in your own projects, download stdlib.unicode.dg and use it in place of the normal stdlib.dg. If you want to see how it's produced, look at the Makefile. That’s where all the magic happens.
